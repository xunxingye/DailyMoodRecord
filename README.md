## 项目概述

这是一个每日心情记录Web应用，用户可以记录每天的心情状态和文字内容，并查看历史记录。

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (原生)
- **后端**: Node.js + Express.js
- **数据库**: MySQL 8.0
- **Web服务器**: Nginx (反向代理)
- **进程管理**: PM2 / systemd

## 系统要求

- **操作系统**: Alibaba Cloud Linux / CentOS 7+ / RHEL 7+
- **内存**: 最低 1GB RAM
- **存储**: 最低 10GB 可用空间
- **网络**: 需要互联网连接用于下载依赖
